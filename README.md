# Support Chatbot MVP

A keyword-based support chatbot built with Node.js and React.

## 🚀 Features
- **Keyword Detection:** Instant responses based on user input.
- **Real-time Interface:** Clean chat UI with message history.
- **Monorepo Structure:** Shared types between frontend and backend.

## 🛠️ Tech Stack
- **Frontend:** React
- **Backend:** Node.js / Express
- **Language:** TypeScript

## 📦 Installation
```bash
npm install
```

## 🚦 Getting Started
Run both the frontend and backend simultaneously:
```bash
npm start
```
*Ensure you have Node.js installed.*
